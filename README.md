# Skin Cancer Classification (EfficientNet-B3)

A minimalist and modular PyTorch repository for skin lesion classification using the HAM10000 dataset.

## Structure
- `data/`: Dataset folder (ignored by git).
- `output/`: Generated logs and model checkpoints.
- `src/`: Source code for training, inference, and data handling.
